# moscow-foodservice-market-analysis

Проект: Исследование рынка заведений общественного питания Москвы

Описание проекта:
Цель проекта — оценить влияние смены шрифтов в мобильном приложении на поведение пользователей и конверсию. Для этого был проведен эксперимент с двумя контрольными группами (A/A) и одной экспериментальной (B). В ходе работы была проанализирована воронка продаж и проверены статистические гипотезы о равенстве показателей между группами.

Ключевые задачи:

Подготовка данных: Очистка данных от дубликатов и неполных записей.
Анализ воронки продаж: Построение последовательности шагов пользователя от входа в приложение до совершения покупки и расчет конверсии на каждом этапе.
Проверка корректности эксперимента: Проверка отсутствия пересечения пользователей между группами.
Статистический анализ: Сравнение ключевых метрик между контрольной и экспериментальной группами с применением поправки на множественную проверку гипотез (метод Шидака).
Формулирование выводов: Определение влияния нового шрифта на поведение пользователей.
Структура проекта:

ab_testing_analysis.py — основной скрипт с кодом для анализа данных и проведения статистических тестов.
data/ — папка для хранения исходного файла logs_exp.csv.
README.md — данный файл с описанием проекта.
